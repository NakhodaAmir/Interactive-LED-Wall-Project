# Overview 
## LED Interactive Wall
### Features
#### LED States
LED Wall Behavior
1) [Bouncing bar](https://github.com/NakhodaAmir/Interactive-LED-Wall-Project/edit/main/README.md#:~:text=states-,AudioTest,-.py)
2) Circle
3) Falling Astroid
4) Percent Volume
5) Rotate
#### LED Controller
Controller state paterns that controls the LED with the above states

# Usage
- Power LED strips(5V & 20A)
- Power the Arduino UNO Q(5-12V & 1-2A)
